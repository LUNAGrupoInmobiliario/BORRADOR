# MaqRent MVP

MVP web para una empresa de renta de maquinaria pesada.

## Funcionalidades
- Dashboard ejecutivo con KPIs.
- Inventario y estatus de maquinaria.
- Registro de clientes.
- Control de rentas activas.
- Cálculo de ingresos estimados.
- Datos persistidos en `localStorage`.
- Diseño responsive, sin dependencias externas.

## Uso
Abre `index.html` en el navegador.

## Alcance MVP
Este prototipo valida la operación comercial y logística. Para producción se recomienda agregar autenticación, base de datos, roles, contratos PDF, evidencias fotográficas, mantenimiento preventivo, facturación e integración de pagos.
